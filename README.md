## Resumen del proyeto
El proyecto es una API con una arquitectura REST de manera completa.

Entre las funcionalidades de la API estan:

-  CRUD.
-  Paginado.
-  Validación. 
-  Filtrar registro.
-  Estandarizar respuestas JSON.
